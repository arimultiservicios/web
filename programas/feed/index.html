Internet Download Manager es compatible con servidores proxy, ftp y protocolos http, firewalls, redirecciones, cookies, autorización, audio MP3 y procesamiento de contenido de video MPEG. IDM se integra perfectamente en Microsoft Internet Explorer, Netscape, MSN Explorer, AOL, Opera, Mozilla, Mozilla Firefox, Mozilla Firebird, Avant Browser, MyIE2 y todos los demás navegadores populares para gestionar automáticamente sus descargas. También puede arrastrar y soltar archivos, o usar el Administrador de descargas de Internet desde la línea de comandos. Internet Download Manager puede marcar su módem a la hora establecida, descargar los archivos que desee y luego colgar o incluso apagar su computadora cuando esté listo.

Otras características incluyen soporte multilingüe, vista previa zip, categorías de descarga, programador pro, sonidos en diferentes eventos, soporte HTTPS, procesador de cola, ayuda html y tutorial, protección antivirus mejorada al finalizar la descarga, descarga progresiva con cuotas (útil para conexiones que usan algún tipo de la política de acceso justo o FAP como Direcway, Direct PC, Hughes, etc.), acelerador de descargas incorporado, y muchos otros.

La versión 6.30 agrega compatibilidad con Windows 10, agrega un panel de descarga de IDM para reproductores web que pueden usarse para descargar videos flash de sitios como MySpaceTV y otros. También cuenta con soporte completo de Windows 8.1 (Windows 8, Windows 7 y Vista), capturador de páginas de videos, programador reconstruido y compatibilidad con el protocolo MMS. La nueva versión también agrega una mejor integración para los navegadores basados ​​en IE 11 e IE, un motor de descarga rediseñado y mejorado, la integración avanzada única en todos los navegadores más recientes, barra de herramientas mejorada y una gran cantidad de otras mejoras y nuevas características.

Other features include multilingual support, zip preview, download categories, scheduler pro, sounds on different events, HTTPS support, queue processor, html help and tutorial, enhanced virus protection on download completion, progressive downloading with quotas (useful for connections that use some kind of fair access policy or FAP like Direcway, Direct PC, Hughes, etc.), built-in download accelerator, and many others.

Version 6.30 adds Windows 10 compatibility, adds IDM download panel for web-players that can be used to download flash videos from sites like MySpaceTV, and others. It also features complete Windows 8.1 (Windows 8, Windows 7 and Vista) support, video page grabber, redeveloped scheduler, and MMS protocol support. The new version also adds improved integration for IE 11 and IE based browsers, redesigned and enhanced download engine, the unique advanced integration into all latest browsers, improved toolbar, and a wealth of other improvements and new features.
